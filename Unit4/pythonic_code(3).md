### 8. 함수 정의하기

- 키워드 def 를 사용하여 함수를 정의, 반드시 들여쓰기 필요
- 독스트링 : 함수 바디의 첫 번째 문장은 문자열 리터럴이 될 수 있음. 이것이 독스트링 → 함수의 설명, help() 또는 docs 생성기에서 읽히는 문장임
- 함수의 실행은 새 지역 심볼 테이블을 만듦. 메모리 안에 함수 전용 공간이 생겨서 함수에서의 모든 변수 대입들은 값을 이곳에 저장함.
- 함수 호출 때 마다 독립된 메모리 공간 생성
- 변수 탐색 순서 : 지역 → 전역 → 내장(print, len, int 등)
- 값에 의한 호출(객체 참조 전달) 방식
- 함수 이름을 현재 심볼 테이블의 함수 객체와 연결

```python
def hello():
		print("Hi")
		
# 메모리 구조
hello -> function object(hello)
```

⇒ 변수에 대입, 인자로 전달, 리스트에 저장, 반환 가능

⇒ 함수 = 객체, 함수 이름 = 객체 가리키는 참조 변수

- return 문이 없는 함수도 None 이라는 값을 돌려줌, print 를 사용하면 확인 가능
